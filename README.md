## Installation
1. Run these following commands (npm audit fix if it shows some third-party pacakages to download):
>npm install
>
>npm audit fix
>
>npm audit

2. Add your own assets folder under src. Update Home.tsx under src/pages with the new asset tags. I have mine under gitignore so you are able to clone and rework this site with your own work. 

3. Run this command to see localhost:
>npm run dev

Or run this command to build your site:
>npm run build
